Jigsaw.js

This Jigsaw was created as a class project.

The user is able to drag and drop objects from one location on the page to another. The grid square that the selected piece is dragged over is then highlighted to help the user identify which grid the piece will be dropped into when the mouse is released. Upon release the image then snaps into that grid piece

A solve it button drops the pieces into the correct grid spaces while a jumble button mixes it all up again.
